## Komentorivitoiminnot

### Sovelluksen koodin suorittaminen

```
mvn compile exec:java -Dexec.mainClass=minesweeper.Main
```

### Testaus

Testit voidaan suorittaa komennolla

```
mvn test
```

Testikattavuusraportti voidaan luoda komennolla

```
mvn jacoco:report
```
